Operadores Relacionais
"São símbolos especiais quais são capazes de realizar
comparações entre determinados operandos e, em seguida,
retornar um resultado"

Tipos:
Similaridade: igual, diferente
Tamanho: maior, maior igual, menor, menor igual

Similaridade
Igualdade: determina se um operando é igual ao outro
Diferença: determina se um operando não é igual ao outro

Símbologia
Igualdade: ==
Diferença: !=

Tamanho
Maior: determina se um operando é maior do que o outro
Maior Igual: determina se um operando é maior ou igual a
outro

Símbologia
Maior: >
Maior Igual: >=

Menor: determina se um operando é menor do que o outro
Menor Igual: determina se um operando é menor ou igual a
outro

Símbologia
Menor: <
Menor Igual: <=
